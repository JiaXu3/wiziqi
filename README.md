# 五子棋
## 功能
有双人对战和人机对战两种模式，下棋时候有倒计时，可以悔棋、重新开始和返回。
功能已经比较完善了，很多bug我也修改好了，功能比较人性话。如果有同学自己还想加一些功能的话我简易可以在开始界面加一张五子棋图片美化界面（得用到sd卡），
或者在给游戏加个背景音乐；0V0：
